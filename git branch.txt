git branch (ver los branches actuales)
git checkout -b newBranch (crear nuevo branch y moverte a el)
git checkout name_of_branch(moverte entre branches)
git push origin hykire (pushea lo que tengas en tu stage al branch que creaste, en mi caso hykire)

Para actualizar al master:
git checkout master
git merge hykire(te mergea lo del branch hykire al master)
